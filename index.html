Audio router

Per-source sends into the shared buses. Rev / Del / Chorus mix knobs still scale wet level; at 0% that effect is silent for everyone. Talk is the phoneme synth; System UI is watch beeps. Breath noise, Cry, and clock tones ride the Main bus (Drive + dry / reverb / delay / chorus). Heartbeat one-shots use the Heartbeat row (enable Dry or Pre-EQ—or both with care—to taste).
